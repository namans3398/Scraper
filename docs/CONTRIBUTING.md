# Contributing

Contributions to Scraper are welcome under the MIT License.

## Issue Reporting

-   Ensure the issue hasn't already been reported.
-   Include steps to reproduce, expected behavior, and actual behavior.
-   Specify your operating system and environment details.

## Pull Requests

1.  Fork the repository.
2.  Create a branch for your feature or fix.
3.  Ensure code is linted and tests pass (`npm run check`).
4.  Submit a pull request detailing the changes and the rationale.
