# Frequently Asked Questions (FAQ)

## What is Scraper?
Scraper is an open-source Electron desktop application that provides a graphical user interface for downloading videos via `yt-dlp`.

## Why did my download fail?
Verify that you have `yt-dlp` installed and updated to the latest version. Website data structures change frequently, requiring `yt-dlp` updates. Check your network connection and ensure the provided URL is publicly accessible.

## Where are my downloads saved?
You can select the destination folder in the user interface before initiating a download.

## Are there any hidden telemetry or trackers?
No. The application is MIT licensed and open source. All source code is visible in the repository.
