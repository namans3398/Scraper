# Code of Conduct

Scraper should be a respectful, practical, and welcoming project for contributors and users.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but accept responsibility for impact.
- Keep discussions focused on the project and the technical work.
- Give clear feedback without personal attacks.
- Respect maintainer decisions about scope, security, platform support, and release readiness.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Sexualized language or imagery.
- Publishing private information without permission.
- Repeatedly derailing issues or pull requests.
- Any conduct that would reasonably make participation unsafe or hostile.

## Reporting

Report unacceptable behavior to the project maintainers through the repository owner's GitHub profile or any private contact channel the maintainers publish. Do not use public issues for sensitive personal reports.

Maintainers may remove comments, close issues, reject pull requests, block users, or take other reasonable action to protect the project.
